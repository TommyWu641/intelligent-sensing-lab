# Intelligent Sensing Lab Website

> A responsive website for an intelligent sensing research lab — academic course project.

## Overview

This website was developed as a coursework project for a web development module. It showcases the Intelligent Sensing Lab's research directions, publications, and recruitment information with interactive features.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, responsive design, print styles)
- JavaScript (ES6)
- No external frameworks — pure vanilla implementation

## Pages & Features

| Page | Description |
|------|-------------|
| Home | Lab overview with interactive elements |
| Research | 6 research directions (AI, IoT, LiDAR, etc.) |
| Publications | Academic papers display |
| Join Us | Application form with validation, print-friendly |
| About | Lab introduction |
| Contact | Contact information |
